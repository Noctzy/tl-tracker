# ⚔️ T&L Tracker

A simple, offline HTML tracker for Throne & Liberty daily, weekly, and monthly tasks.

> **Note:** This tool was built with AI assistance and is primarily for personal use.

## ✨ Features

- 📅 Daily, weekly, and monthly task checklists
- ⏱️ Custom countdown timers (like Amitoi expeditions)
- 📊 Progress bars for each category
- 🔄 Auto-reset at **08:00 UTC**
- 🏷️ Counter support for multi-step tasks (e.g., 3/3 dungeons)
- 👤 Multiple profiles (up to 6 characters)
- 📝 Personal notes per profile
- 🎨 3 themes: Dark, Light, T&L (purple/gold)
- 💾 All data saved locally - no accounts, no tracking, no cloud sync

## 🚀 How to Use

1. Download the `TL-Checklist.html` file
2. Double-click to open in your browser
3. Check off tasks as you complete them
4. Everything saves automatically to your browser's local storage

## 🔒 Privacy

- **No data is ever sent anywhere**
- All progress stays in your browser
- **No accounts, no tracking, no analytics**
- Completely offline after download
